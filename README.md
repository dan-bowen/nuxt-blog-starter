## Quickstart

Start the Nuxt App

```shell
cd nuxt-app
yarn install
yarn dev
```

Start the mock server.

```shell
cd devtools
yarn install
yarn mocks
```

> Read more about the [Mocks Server](https://www.mocks-server.org/).
